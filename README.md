# AA Translator — ASCII Art 번역기

Gemini AI를 활용한 AA(아스키 아트) 파일 번역기입니다.  
파일을 업로드하고, 텍스트를 선택하고, 버튼 하나로 번역할 수 있습니다.

---

## 🚀 빠른 시작 (초보자용)

### 1단계: Node.js 설치
아래 링크에서 **Node.js**를 다운로드하고 설치하세요 (LTS 버전 권장):  
👉 [https://nodejs.org](https://nodejs.org)

### 2단계: 실행하기

**Windows:**
- `start.bat` 파일을 **더블클릭**하세요.
- 처음 실행 시 자동으로 필요한 파일이 설치됩니다.
- 브라우저가 자동으로 열립니다.

**Mac / Linux:**
```bash
./start.sh
```

### 3단계: API Key 입력
- 앱이 열리면 **API Key 입력 창**이 자동으로 나타납니다.
- [Google AI Studio](https://aistudio.google.com/app/apikey)에서 무료로 API Key를 발급받아 입력하세요.
- 키는 브라우저에만 저장되며, 한번 입력하면 다시 입력할 필요 없습니다.

> 💡 **API Key 발급 방법:**  
> 1. https://aistudio.google.com/app/apikey 접속  
> 2. Google 계정으로 로그인  
> 3. "Create API Key" 클릭  
> 4. 생성된 키를 복사하여 앱에 붙여넣기

### 끝!
파일을 업로드하고 번역하면 됩니다! 🎉

---

## 📖 수동 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:3000 을 열어주세요.

---

## 사용법

1. `.txt` / `.mlt` / `.ast` 파일을 업로드하거나 드래그 앤 드롭
2. 번역할 텍스트를 선택
3. 번역 버튼 클릭
4. 번역된 파일 다운로드
